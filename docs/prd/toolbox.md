## 功能清单

### 第一批次（本期实现）

#### 1. 开发工具 Tab
- [ ] **JSON 格式化工具**
  - 格式化：将压缩的 JSON 格式化为可读形式（带缩进）
  - 压缩：去除所有不必要的空白字符
  - 验证：检查 JSON 语法合法性，显示错误位置
  - 转义/去转义：将 JSON 字符串进行转义（如 `"` → `\"`）或反向操作
  - 折叠/展开：代码折叠功能，支持展开/收起对象和数组
  - 输入方式：文本框直接输入、粘贴、拖拽文件上传
  - 复制结果：一键复制处理后的结果到剪贴板
  - 清空：一键清空输入内容

- [ ] **图片 Base64 互转工具**
  - 图片 → Base64：上传图片文件，输出 Base64 编码
  - Base64 → 图片：输入 Base64 字符串，预览并下载图片
  - 输入方式：拖拽上传、点击选择文件、粘贴剪贴板图片
  - 输出格式选项：
    - Data URI 格式（`data:image/png;base64,...`）
    - 纯 Base64 字符串
  - 图片预览：转换后实时预览图片
  - 复制结果：一键复制 Base64 结果
  - 下载图片：从 Base64 下载原图
  - 文件信息：显示文件名、大小、尺寸、格式

## 技术实现建议

### 纯前端实现
- 无需后端接口，所有逻辑在浏览器完成
- JSON 处理使用原生 `JSON.parse`/`JSON.stringify`
- 图片处理使用 `FileReader` API 读取文件
- 剪贴板操作使用 `navigator.clipboard` API

### 代码折叠方案（JSON 工具）
**方案 A（推荐）**：使用 Monaco Editor（VS Code 同款编辑器）
- 优点：功能强大，自带折叠、语法高亮、错误提示
- 缺点：包体积较大（约 1MB+），需要按需加载

**方案 B**：使用 CodeMirror 6
- 优点：轻量，插件丰富
- 缺点：配置相对复杂

**方案 C**：简单实现（如无编辑器库）
- 仅使用 textarea + 行号显示
- 折叠功能用缩进视觉提示代替
- 或本期暂不提供折叠功能

### 图片处理
- 文件读取：`FileReader.readAsDataURL()`
- 图片信息：创建 `Image` 对象获取尺寸
- 下载功能：创建 `a` 标签触发下载

### 状态管理
- 无需 Pinia Store（无持久化需求）
- 每个工具页面使用本地 state 管理
- 刷新页面即清空所有数据

## 验收标准

### JSON 格式化工具

**Given** 用户在 JSON 工具页面
**When** 在输入框粘贴一段压缩的 JSON
**And** 点击 "格式化" 按钮
**Then** 输出框显示带缩进的格式化 JSON

**Given** 用户在 JSON 工具页面
**When** 在输入框输入无效的 JSON
**And** 点击 "验证" 按钮
**Then** 显示错误提示，指出语法错误位置

**Given** 用户在 JSON 工具页面
**When** 在输入框粘贴 JSON
**And** 点击 "转义" 按钮
**Then** 输出框显示转义后的字符串（引号前带反斜杠）

**Given** 用户在 JSON 工具页面
**When** 处理完成后点击 "复制结果"
**Then** 结果内容被复制到剪贴板，并显示成功提示

### 图片 Base64 工具

**Given** 用户在图片 Base64 工具页面
**When** 拖拽一张图片到拖拽区域
**Then** 图片被成功读取，显示预览和文件信息
**And** Base64 结果自动显示在输出框

**Given** 用户在图片 Base64 工具页面
**When** 复制一张图片到剪贴板
**And** 在页面按 Ctrl+V
**Then** 图片被成功读取并转换

**Given** 用户在图片 Base64 工具页面
**When** 选择 "纯 Base64" 输出格式
**Then** 输出结果不包含 `data:image/...` 前缀

**Given** 用户在图片 Base64 工具页面，处于 Base64→图片模式
**When** 粘贴一段有效的图片 Base64
**Then** 图片被成功解码并显示预览

**Given** 用户在图片 Base64 工具页面
**When** 点击 "下载图片" 按钮
**Then** 浏览器触发下载，保存原图文件

## 依赖建议

### 可选依赖（JSON 编辑器）
- `monaco-editor` - VS Code 编辑器（功能最全）
- `@codemirror/lang-json` - CodeMirror 6 JSON 支持（轻量）

### 无需额外依赖
- 图片处理：原生 FileReader API
- 剪贴板：原生 Clipboard API
- JSON 处理：原生 JSON API

---

## 性能考虑

- 大图片处理：建议限制上传图片大小（如 5MB），避免浏览器卡顿
- JSON 大文件：超过 1MB 的 JSON 文件处理时显示加载提示
- 防抖处理：输入框内容变化时防抖处理，避免频繁转换

---

## 安全考虑

- 图片转换仅在客户端完成，不上传服务器
- 不存储用户处理的任何数据
- 刷新页面即清空所有内容

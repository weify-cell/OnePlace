# 哈希计算工具设计方案

## 概述

百宝箱工具中的哈希计算工具，支持 MD5、SHA1、SHA256、SHA512、SHA-3 系列、SM3 等多种哈希算法，提供文本输入和文件上传两种模式，包含哈希值计算和校验功能。

## 页面布局

```
┌─────────────────────────────────────┐
│  标题：哈希计算                      │
├─────────────────────────────────────┤
│  [MD5/SHA1] [SHA2] [SHA-3/SM3]       │  ← 标签切换
├─────────────────────────────────────┤
│  ● 文本输入  ○ 文件上传              │  ← 输入方式切换
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐    │
│  │ 文本输入框 / 文件拖拽区       │    │
│  └─────────────────────────────┘    │
│  [计算哈希]                         │
├─────────────────────────────────────┤
│  计算结果：                         │
│  ┌─────────────────────────────┐    │
│  │ 哈希值显示区（可复制）        │    │
│  └─────────────────────────────┘    │
│  ─────────── 分割线 ───────────     │
│  校验：输入预期哈希值 [校验] [结果]  │
└─────────────────────────────────────┘
```

## 分标签算法分布

| 标签 | 算法 |
|------|------|
| MD5/SHA1 | MD5、SHA1 |
| SHA2 | SHA256、SHA512 |
| SHA-3/SM3 | SHA3-256、SHA3-512、SM3 |

## 交互流程

### 文本输入模式

1. 选择算法标签
2. 选择"文本输入"模式
3. 输入文本 → 点击"计算哈希" → 显示结果
4. 可在页面下方校验框输入预期哈希值验证

### 文件上传模式

1. 选择算法标签
2. 选择"文件上传"模式
3. 拖拽或点击上传文件
4. 显示文件名、大小、进度条
5. 计算完成后显示哈希值

## 状态设计

| 状态 | 显示内容 |
|------|---------|
| 空状态 | 提示文字：请输入文本或上传文件 |
| 计算中 | 进度条（文件模式）/ 加载指示（文本模式） |
| 计算完成 | 哈希值 + 一键复制按钮 |
| 校验匹配 | 绿色"匹配"提示 |
| 校验不匹配 | 红色"不匹配"提示 |

## 技术方案

### 依赖库

| 算法 | 库 |
|------|-----|
| MD5 | md5 |
| SHA1 | sha.js |
| SHA256/SHA512 | sha.js |
| SHA3-256/SHA3-512 | js-sha3 |
| SM3 | sm3 |

### 文件处理

- FileReader + 分片读取（大文件不阻塞 UI）
- 进度显示：FileReader 的 onprogress 事件

### 组件结构

- 新建 `src/views/tools/HashToolView.vue`
- 复用 `src/components/toolbox/ToolLayout.vue`

### 路由配置

- 路由路径：`/toolbox/hash`
- 需在 `src/router/index.ts` 添加路由
- 需在 `src/components/toolbox/tools.config.ts` 更新工具配置（status: 'available'，routePath: '/toolbox/hash'）

## 实现步骤

1. 安装依赖：md5、sha.js、js-sha3、sm3
2. 创建 `HashToolView.vue` 组件
3. 配置路由
4. 更新 tools.config.ts
5. 添加单元测试

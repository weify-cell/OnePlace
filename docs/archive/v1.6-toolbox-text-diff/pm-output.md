## 需求摘要

### 核心需求
在百宝箱模块中新增文本比较工具，提供类似 VS Code diff 的逐字符比较能力，适合文案校对场景。

### 方案选定
**方案 B（已选定）**：逐字符比较
- 类似 VS Code diff，精细到字符级别
- 适合文案校对、敏感词检查等场景
- 差异高亮显示，新增/删除区分颜色

---

## 功能范围

### 新增功能

#### 文本比较 Tab（/toolbox/text-diff）
1. **双栏输入**：左侧原文、右侧对比文，支持手动输入和粘贴
2. **逐字符 diff**：逐字符比较，高亮显示每个差异
3. **差异类型区分**：
   - 新增字符：绿色背景
   - 删除字符：红色背景 + 删除线
   - 相同字符：正常显示
4. **差异导航**：上一处/下一处差异按钮
5. **统计信息**：差异处总数、相同字符数
6. **复制报告**：一键复制含位置信息的差异报告
7. **清空功能**：分别清空或一键清空全部
